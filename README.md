# 机读卡答题区域定位系统

针对标准化机读卡识别场景，设计并实现了一套基于几何变换与形态学算法的答题区域定位系统，解决了因图像倾斜、印刷偏差导致的区域定位不准问题。

## 功能特性

- 图像矫正
- 答题区域定位（水平和垂直）
- 答案输出
- 图形用户界面（GUI）

## 安装

1. 克隆仓库：
   ```
   git clone <repository-url>
   cd -
   ```

2. 安装依赖项：
   ```
   pip install -r requirements.txt
   ```

## 依赖项

- numpy
- Pillow
- opencv-python

## 使用方法

运行主程序：
```
python main.py
```

在 GUI 中：
1. 选择“文件” > “打开图像”加载机读卡图像。
2. 使用“编辑”菜单进行图像矫正、区域定位等操作。
3. 最终输出答案。
